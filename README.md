# Sample / Webpack v5


This is sample for webpack v5 that is using babel & typescript & react & react-refresh

| feature    | tool          |
|------------|---------------|
| Installer  | yarn          |
| Transpile  | babel         |
| Typecheck  | typescrpt     |
| Components | react         |
| HMR        | react-refresh |


## Tasks

### development

```
yarn run start
```

### build

```
yarn run build
```
